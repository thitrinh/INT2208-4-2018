Cách cài đặt phần mềm:
Dùng xampp.
1. Tạo database có tên "memrise" 
 + trong database tạo 2 bảng tên là "account" và "vocabulary"
 account gồm (id (khóa chính, auto_increment), username, password, email);
 vocabulary gồm (id (khóa chính, auto_increment), word, mean, subject);
 + chạy file 1.html.
2. Tạo tài khoản ngườ dùng.
Người dùng đầu tiên tạo tài khoản sẽ có admin có thể quản trị trang web.
Admin sẽ vào trang để thêm từ.
Người tạo tài khoản tiếp theo sẽ là người dùng.
